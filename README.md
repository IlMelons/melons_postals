# Melons Postals
Simple script that allows player to set waypoints to specific postal codes with `/gps`

## Dependencies & Start Order
- ensure `ox_lib`
- ensure `Framework`
- ensure `melons_postals`

## Command
To set a Waypoint to a certain postal, type `/gps [postalCode]`
